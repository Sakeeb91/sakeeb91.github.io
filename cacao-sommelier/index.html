Cacao. | Modern Chocolatier

Premium chocolate e-commerce with AI-powered features

Overview

A modern, premium chocolate e-commerce website that combines elegant design with cutting-edge AI features. Built with React and TypeScript, the site showcases a sophisticated user experience with glassmorphism effects, smooth animations, and AI-powered functionality powered by Google's Gemini API.

Live Demo Source Code

Technology Stack

React TypeScript Vite Tailwind CSS Google Gemini API Cloudflare Pages

Key Features

  • AI-Powered Pairing Suggestions: Gemini API generates sophisticated chocolate pairing recommendations with search grounding
  • Text-to-Speech Tasting Notes: Audio generation for immersive sensory descriptions
  • Dynamic Image Generation: AI-generated product imagery using Gemini Flash Image model
  • Modern Glassmorphism UI: Premium design with backdrop blur effects and smooth animations
  • Responsive Design: Fully responsive layout optimized for all devices
  • Global CDN Deployment: Hosted on Cloudflare Pages for maximum performance

Technical Highlights

  • Performance: Built with Vite for lightning-fast development and optimized production builds
  • Type Safety: Full TypeScript implementation for robust code quality

  • Deployment: Automated CI/CD pipeline with GitHub integration
  • SEO Optimized: Proper meta tags, semantic HTML, and performance optimization

Development Process

This project involved fixing critical rendering issues, implementing graceful error handling for API failures, and deploying to Cloudflare Pages via CLI. The development process included:

  • Debugging and fixing React rendering issues
  • Implementing conditional API initialization to prevent crashes
  • Setting up environment variables for secure API key management
  • Configuring Cloudflare Pages deployment with Wrangler CLI
  • Optimizing build process and bundle size
