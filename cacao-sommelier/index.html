Cacao. | Modern Chocolatier

Premium chocolate e-commerce with AI-powered features

A modern, premium chocolate e-commerce website that combines elegant design with cutting-edge AI features. Built with React and TypeScript, the site showcases a sophisticated user experience with glassmorphism effects, smooth animations, and AI-powered functionality powered by Google's Gemini API.

View Live Demo View Source Code

Technology Stack

React 18 TypeScript Vite Tailwind CSS Google Gemini API Cloudflare Pages Framer Motion

The landing page features a striking hero section with "The Physics of Pleasure" tagline.

Key Features

AI Pairing Sommelier

Gemini API generates sophisticated chocolate pairing recommendations with search grounding, offering expert advice for any food or drink.

Sensory Audio Guides

Text-to-speech generation creates immersive tasting notes, guiding users through the flavor profile of each chocolate.

Dynamic Imagery

AI-generated product imagery using Gemini Flash Image model creates unique, high-quality visuals for the shopping experience.

Glassmorphism UI

Premium design aesthetic with backdrop blur effects, smooth transitions, and a sophisticated color palette.

Technical Highlights

  • Performance First: Built with Vite for lightning-fast development and optimized production builds with code splitting.
  • Type Safety: Comprehensive TypeScript implementation ensuring robust code quality and developer experience.
  • Secure Architecture: Environment variable handling for API keys and secure deployment practices.
  • CI/CD Pipeline: Automated deployment to Cloudflare Pages via GitHub actions/webhooks.
